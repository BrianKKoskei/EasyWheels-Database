# EasyWheels-Database

A relational database project designed to manage a car dealership system, tracking inventory, sales, and employee roles.

## Project Structure
To set up this database, execute the files in the following order:

1. *1_Tables.sql*: Contains the schema definition (CREATE TABLE statements).
2. *2_DummyData.sql*: Populates the tables with test records for demonstration.
3. *3_Queries.sql*: Includes various analytical queries for reporting and data filtering.
4. *4_Triggers&Procedures.sql*: Contains stored procedures and automated triggers to handle business logic.

## Prerequisites
* MySQL or a similar RDBMS.
* A database management tool (e.g., DBeaver or MySQL Workbench).
